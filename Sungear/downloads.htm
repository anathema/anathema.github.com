Quickstart Guide

Requirements

Anathema is written in Java and will run on any system with a compatible Java Virtual Machine. This includes Windows, Linux and Mac OS X. Make sure that Java Runtime Environment Version 5.0 or later is installed.

We have found these system specs sufficient for running Anathema:

  • 1.6 GHz CPU
  • 256 MB RAM
  • 20 MB available hard disk space

If your results differ, please let us know.

Download

Go to the SourceForge.net download system for a list of file releases. The most recent release is listed at the top. Choose the file most suited for your operating system, or the "raw" .ZIP archive for a generic distribution.

Should the release be designated as a "patch", you will need to download and install the respective regular release first.

Installation

Regular releases are pre-packed and installed on their own. Auto-installing packages for different operating systems are available. If you chose to download the raw package, the archive can simply be extracted into the target directory.

Patches need to be applied to an existing installation to work. To this end, the patch-archive has to be extracted over the existing installation of the corresponding regular release.

Launch

Depending on your download choice and operating system, you are provided with a executable file or link for launching Anathema. If you chose the raw download, chances are your system allows to launch Anathema by double-clicking "anathema.jar" in the main folder. Failing that, you need to open a console window, switch to the installation directory and type "java -jar anathema.jar".

Have fun with Anathema!

Sourcecode Downloads

The latest development code resides in the "trunk"-folder of our Subversion (SVN) repository at all times.

You can download a specific release's code using its tag in the "tags" folder.

If you want to browse Anathema's source code online, use the Fisheye repository browser.
