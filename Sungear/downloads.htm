Quickstart Guide

The following is a quick step by step guide to help you download and install Anathema.

Requirements

Before you download Anathema, make sure your system meets the following requirements to ensure optimal performance.

Anathema is written in Java and will run on any system with a compatible Java Virtual Machine, that includes Windows, Linux AND Macintosh. Make sure that you have the Java Runtime Environment Version 5.0 installed on your system
  • 500MHz CPU
  • 128MB System RAM
  • 300MB free Harddrive space

Download

Download the latest files for Anathema from the SourceForge download system.

The newest files are always conveniently listed at the top. Simply download all neccessary files with the word "user" in their name. These files come in two versions, each with a slightly different installation procedure.

Full Releases, indicated by some Exalted-related name. These releases are installations that stand alone.
Patches, clearly designated by the word "patch" in the file name. These fix and improve minor issues in already installed releases.

Installation

Full Releases are pre-packed and installed on their own. There are different variations avilable for different operating systems. Depending on the package downloaded, the archive can either simply be extracted into the target directory, or the supplied installation script (some operating systems only) can be executed to easily and properly install Anathema.

Patches, on the other hand, need to be properly applied to an existing installation to work. In the case of archives, they have to be extracted over an existing installation of the corresponding regular release to work properly. Some patches can also come as self-extracting installers that work almost automatically.

Note also that Anathema is not yet feature complete and that all packages should only be considered preview releases.

Sourcecode Downloads

Each Full Release is also available as a "developer" archive, a simple package that contains everything in the user package together with the program source code. We will also offer this service for Patches archives whenever possible. Just like the "user" installations, these files can be found on the SourceForge download system.
