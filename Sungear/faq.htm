Technicalities

What hardware do I need to run Anathema?

See the Downloads page for a detailed list of hardware requirements.

What software do I need to run Anathema?

The Anathema regular releases come with everything you need, except for the Java Runtime Environment. Patch releases require the appropriate regular release to be installed.

Java is installed, but Anathema still won't run.

Make sure you upgrade your installation to Java 5.0 before running Anathema.

I'd like further instructions. Any chance?

Of course. A basic installation guide can be found on the Downloads page, and an extended Installation Guide is available for download on the SourceForge project page.

Bugs & Features

When will rule/feature X be supported?

It's hard to plan ahead more than two releases, and even then there are just as many things that simply "happen to be done along the way", as there are those that "aren't that important right now".
That said, have a look at the feature requests page - if your feature isn't listed there, just submit it and wait for a comment from us within the next day or so.

Things don't work right! What should I do?

If you found a bug, be sure to post it over at the bug report page! We try to fix all bugs as soon as possible, and you can make things much easier for us if you give a concise description of what happened. If, in addition to finding the bug, you also know how to fix it (or just think you know), let us know!

Contribute

How can I contribute to Anathema?

The easiest way is giving feedback. Just let us know what you think, especially what you think could be done better.
For more in-depth-work, you could come up with documentation or additional art/icons for the program or expand our database. If you have an idea, but aren't certain it's suitable, just contact us.
