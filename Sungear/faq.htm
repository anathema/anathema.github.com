Technicalities

What hardware/software do I need to run Anathema?

Anathema will run on any operating system with Java Runtime Environment Version 5 or later, including Linux, Windows and Mac OS X.

Patch releases require the appropriate regular release to be installed.

See the Downloads page for an estimate of hardware requirements.

Java is installed, but Anathema still won't run.

First, ensure that you've extracted all ZIPped files to their full pathnames.

If errors persist, please ascertain that you have the latest release.

Should all that fail, please provide the following information when contacting us:

  • Which version of java are you using?
  • What is your operating system?
  • Which release of Anathema causes the error?
  • What command or action do you use to launch Anathema?
  • Which plugins are installed?

I'd like further instructions. Any chance?

Of course. A quickstart guide can be found on the Downloads page, while an extended installation guide is available for download on the project page.

Bugs & Features

When will rule/feature X be supported?

It's hard to plan ahead more than two releases, and even then there are just as many things that simply "happen to be done along the way", as there are those that "aren't that important right now".
That said, have a look at the feature requests page - if your feature isn't listed there, just submit it and wait for a comment from us within the next day or so.

Things don't work right! What should I do?

If you found a bug, be sure to post it over at the bug report page! We try to fix all bugs as soon as possible, and you can make a difference by giving a concise description of what happened.
