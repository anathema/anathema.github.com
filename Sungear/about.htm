What is Anathema?

Anathema is an all-round toolkit for White Wolf's Exalted, offering a multitude of features for players and storytellers alike. These include:

  • Character creation and management
  • Series management
  • PDF export for both of the above
  • Extensive visual charm cascades
  • A music database

For more information and screenshots, please see the Features page.

Anathema Development

Anathema is free software developed by fans and enthusiasts of the Exalted game. Everyone working on the project does so in their spare time and without compensation. If you want to support the program please consider donating.
