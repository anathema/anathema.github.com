Website Use Guidelines

Only use the following tags:

  • h2 (section overview with underline)
  • h3 (sub-header)
  • p (paragraph)
  • ul with li (lists)
  • blockquote (bordered and indented. Use for image)
  • br (linebreak)
  • b (bold)

Images should always be 175px wide, so that 3 of them fit nicely next to each other within one single line using a blockquote tag.
