Character Management

Character Management, as easy as never before. Already featuring most rules from the core book as well as Charms from most of the supplements, Anathema allows you to create and maintain characters on the fly. With fully graphical Charm selection, a sidebar with all important data and an optimizing calculator for Bonus Points, you'll be done in no time. Use the PDF-Export feature to print faithful reconstructions of the classic White Wolf character sheet as well as Voidstate's widely acclaimed sheet, and you're set to play. All of the five "core" Exalted as well as (Heroic) Mortals are supported.
